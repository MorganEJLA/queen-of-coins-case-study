# Queen of Coins — Case Study

A case study page for [Queen of Coins](https://github.com/MorganEJLA/QueenOfCoins), a tarot-anchored daily ritual app built with React Native and Expo.

---

## About

This page documents the origin, design decisions, technical architecture, and feature set behind Queen of Coins — a personal project built to solve a real problem: creating structure and ritual after leaving a career with an externally imposed schedule.

---

## Built With

- Vanilla HTML, CSS, JavaScript
- No frameworks or libraries
- Hosted on GitHub Pages
